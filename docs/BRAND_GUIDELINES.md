## Brand Overview

### Mission

Gynergy empowers individuals to transform their lives through guided daily practices, community support, and the science of habit formation.

### Brand Personality

- **Empowering** - We believe in your potential
- **Supportive** - We're with you every step
- **Authentic** - Real progress, real people
- **Warm** - Approachable and human
- **Focused** - Clear, intentional guidance

### Core Values

1. **Progress over perfection** - Small steps compound
2. **Community strength** - Together we achieve more
3. **Science-backed** - Evidence-based practices
4. **Self-compassion** - Kindness in the journey

## Logo

### Primary Logo

The Gynergy logo represents growth, energy, and transformation. It should always be displayed with adequate clear space.

### Clear Space

Maintain minimum clear space equal to the height of the "G" around all sides of the logo.

### Logo Variations

| Variation   | Usage                              |
| ----------- | ---------------------------------- |
| Full color  | Primary usage on light backgrounds |
| White       | On dark backgrounds                |
| Teal/Action | Monochrome applications            |
| Icon only   | App icons, favicons                |

- âŒ Don't change the logo colors
- âŒ Don't use the logo smaller than 24px height

## Color Palette

### Primary Colors

#### Action (Teal)

**#99f6e0** - Our signature color

- Use for: CTAs, links, interactive elements
- Conveys: Energy, growth, possibility

#### Primary (Orange)

**#ffc878** - Warmth and achievement

- Use for: Streaks, achievements, highlights
- Conveys: Warmth, accomplishment, celebration

#### Purple

**#a86cff** - Distinction and creativity

- Use for: Badges, admin roles, accents
- Conveys: Special status, creativity

### Secondary Colors

| Color   | Hex     | Usage                                 |
| ------- | ------- | ------------------------------------- |
| Success | #7ef698 | Completed states, positive feedback   |
| Danger  | #fd6a6a | Errors, warnings, destructive actions |
| Warning | #f69859 | Caution, pending states               |

### Background Colors

| Light Mode | Dark Mode | Usage                   |
| ---------- | --------- | ----------------------- |
| #ffffff    | #131313   | Primary background      |
| #f5f5f5    | #27282a   | Card/surface background |

### Color Accessibility

All color combinations must meet WCAG 2.1 AA standards:

- **Normal text**: 4.5:1 contrast ratio minimum
- **Large text (18px+)**: 3:1 contrast ratio minimum
- **UI components**: 3:1 contrast ratio minimum

### Color Do's and Don'ts

**Do:**

- âœ… Use teal (#99f6e0) for primary actions
- âœ… Use orange (#ffc878) for achievements
- âœ… Maintain sufficient contrast ratios
- âœ… Use semantic colors for states (success, danger)

**Don't:**

- âŒ Use red for non-error states
- âŒ Use color as the only indicator of meaning
- âŒ Use low-contrast color combinations
- âŒ Override brand colors with arbitrary values

## Typography

### Primary Font

**Work Sans**

A grotesque sans-serif typeface that's professional yet approachable. It offers excellent readability at all sizes and weights.

```css
font-family: "Work Sans", system-ui, sans-serif;
```

### Type Scale

| Purpose | Size | Weight   | Line Height |
| ------- | ---- | -------- | ----------- |
| Hero    | 48px | Bold     | 1.1         |
| H1      | 48px | Bold     | 1.2         |
| H2      | 36px | Semibold | 1.2         |
| H3      | 24px | Semibold | 1.3         |
| Body    | 18px | Regular  | 1.5         |
| Small   | 16px | Regular  | 1.5         |
| Caption | 14px | Regular  | 1.4         |

### Typography Guidelines

**Do:**

- âœ… Use sentence case for headlines
- âœ… Keep line lengths to 60-80 characters
- âœ… Use adequate line spacing (1.5 for body)
- âœ… Left-align body text

**Don't:**

- âŒ Use ALL CAPS for more than 2-3 words
- âŒ Use light font weights below 18px
- âŒ Mix more than 2 font weights in one context
- âŒ Center-align long paragraphs

## Voice & Tone

### Brand Voice

Gynergy speaks with **encouragement, clarity, and authenticity**.

### Voice Characteristics

| Characteristic | We Are                     | We're Not             |
| -------------- | -------------------------- | --------------------- |
| Empowering     | "You've got this"          | Preachy or lecturing  |
| Supportive     | "We're here with you"      | Judgmental            |
| Authentic      | "Real progress takes time" | Fake or hyped         |
| Clear          | "Here's what to do next"   | Vague or jargon-heavy |
| Warm           | "Welcome back, Sarah"      | Cold or corporate     |

### Tone Variations

**Celebration**

> "Amazing! You've completed a 7-day streak. Your consistency is inspiring! ğŸ”¥"

**Encouragement**

> "Progress isn't always linear. Taking a break doesn't erase your growth."

**Instruction**

> "Find a quiet space, close your eyes, and take three deep breaths."

**Community**

> "Your cohort is cheering you on. 12 members shared encouragement today."

### Writing Guidelines

**Do:**

- âœ… Use "you" and "your" (direct address)
- âœ… Write in active voice
- âœ… Keep sentences concise
- âœ… Use specific numbers when possible
- âœ… Celebrate small wins

**Don't:**

- âŒ Use guilt or shame ("You failed to...")
- âŒ Overpromise results
- âŒ Use excessive exclamation marks
- âŒ Sound robotic or impersonal
- âŒ Use jargon without explanation

### Example Copy

| Context | Good                                | Avoid                    |
| ------- | ----------------------------------- | ------------------------ |
| Login   | "Welcome back, Sarah"               | "User login successful"  |
| Streak  | "12 days strong! ğŸ”¥"                | "Streak: 12"             |
| Error   | "Couldn't save. Let's try again."   | "Error 500: Save failed" |
| Empty   | "Your journey begins with one step" | "No data available"      |

## Imagery Style

### Photography Guidelines

**Subject Matter:**

- Real people in authentic moments
- Diverse representation
- Natural settings (nature, home)
- Calm, focused environments

**Visual Treatment:**

- Warm, natural lighting
- Soft, inviting color tones
- Minimal post-processing
- Avoid overly staged shots

**Themes:**

- Morning routines
- Meditation and mindfulness
- Journaling and reflection
- Nature and growth
- Community connection

### Illustration Style

- Clean, minimal line work
- Warm color palette
- Nature-inspired motifs (leaves, sun, growth)
- Approachable, not overly polished

### Iconography

- Use the Gynergy icon font (`gng-*` prefix)
- Consistent stroke weight
- Simple, recognizable shapes
- 24px standard size

- Focus trapped within modal
- Close on Escape key
