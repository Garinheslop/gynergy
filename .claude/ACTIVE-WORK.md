## Quick Rules

### Before Starting Work

1. Check this dashboard - is another developer active?
2. If Active: Coordinate before modifying shared files
3. If Inactive: Safe to proceed, update your tracker

### During Work

- Update your personal tracker
- Lock files you are actively modifying
- Commit and push frequently

### After Work

- Set your status to Inactive
- Update handoff notes in your tracker
- Push all changes to remote
